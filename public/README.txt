Pon aqui tus canciones en mp3.

Ejemplos:
- public/la-correcta.mp3
- public/no-se-va.mp3

Despues cambia el campo audioUrl de cada cancion en src/App.tsx por:
- /la-correcta.mp3
- /no-se-va.mp3

Para cambiar fotos del sitio:
1. Guarda tus imagenes en src/assets/images
2. Puedes reemplazar estos archivos manteniendo el mismo nombre:
- birthday_cake_26_1780576134041.png
- romantic_love_letter_1780576148270.png
- memories_polaroid_path_1780576163187.png

Si prefieres usar nombres nuevos:
1. Copia la imagen a src/assets/images
2. Cambia el import correspondiente en src/App.tsx
3. Haz push otra vez
